# Win Focus Mouseover

## Python variant

- Requires python 3.9 installed
- Install pywin32
  - `pip install pywin32`
- Run with:
  - from project root dir
  - `python3 mouseover.py`

## cpp variant

- Any cmake supported cpp compiler for win32
- cmake
- build
  - from project root dir
  - `cmake -B build`
  - `cmake --build build`
- Run with
  - from project root dir
  - `build\Release\MouseOver.exe`
